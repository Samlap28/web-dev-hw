My design will be about sustainability. I will include links to
some startups that revolve around environmental sustainability.
With this assignment, I hope to learn how to create functional websites
so that I can do my part with environmental, sustainable programs. In my
design, I will include a site logo with a size element of 200px. Padding element
of 20-50px for page contect. Margin element 10-30px between site logo and the image
content. A display element of block for the page contect. A pseudo class hover effect
for the links. And a fixed position for a majority of the elements on the page.
